# DigiTools Platform

A modern digital marketplace UI where users can explore premium tools, add products to cart, manage checkout flow, and experience responsive sections for pricing, onboarding, and call-to-action.

## Technology Used

- React
- Vite
- Tailwind CSS
- DaisyUI
- React Toastify

## Key Features

1. Interactive product section with toggle between Products and Cart views.
2. Cart management system with add/remove functionality, total calculation, and checkout clear action.
3. Modern responsive landing page sections including hero, pricing plans, step-by-step onboarding, CTA banner, and social footer.
